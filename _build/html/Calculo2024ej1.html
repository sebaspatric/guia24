Pregunta1: limite sucesión#

Encontrar el límite de la sucesión: $\( a_n = \frac{4 + \frac{1}{n} + 4 + \frac{2}{n} + \cdots + 4 + \frac{n}{n}}{n} \)$

Desarrollo#

Cada término del numerador es de la forma

\[ 4 + \frac{k}{n}, \quad k = 1,2,\dots,n. \]

Por lo tanto, el numerador se puede escribir como:

\[ n a_n = \sum_{k=1}^{n} \left( 4 + \frac{k}{n} \right). \]

Ahora separamos las sumas:

\[ n a_n = \sum_{k=1}^{n} 4 + \sum_{k=1}^{n} \frac{k}{n}. \]
\[ n a_n = 4n + \frac{1}{n} \sum_{k=1}^{n} k. \]

Usando la fórmula de la suma de los primeros \(n\) enteros:

\[ n a_n = 4n + \frac{1}{n} \cdot \frac{n(n+1)}{2}. \]
\[ n a_n = 4n + \frac{n+1}{2}. \]

Finalmente, dividiendo entre \(n\):

\[ a_n = \frac{4n + \tfrac{n+1}{2}}{n}. \]
\[ a_n = 4 + \frac{n+1}{2n}. \]

Cálculo del límite#

\[\begin{split} \begin{aligned} \lim_{n \to \infty} a_n &= \lim_{n \to \infty} \left( 4 + \frac{n+1}{2n} \right) \\[6pt] &= 4 + \lim_{n \to \infty} \frac{n+1}{2n} \\[6pt] &= 4 + \lim_{n \to \infty} \left( \frac{n}{2n} + \frac{1}{2n} \right) \\[6pt] &= 4 + \lim_{n \to \infty} \left( \tfrac{1}{2} + \tfrac{1}{2n} \right) \\[6pt] &= 4 + \left( \tfrac{1}{2} + 0 \right) \\[6pt] &= 4 + \tfrac{1}{2} \\[6pt] &= \tfrac{9}{2}. \end{aligned} \end{split}\]

Respuesta:#

d) \(\lim a_n = \dfrac{9}{2}\)

#código para gráfica interactiva de la sucesión a_n
import numpy as np
import plotly.graph_objects as go

# Valores iniciales
n_max_init = 100
const_init = 4

n_values = np.arange(1, n_max_init+1)
a_n = const_init + (n_values + 1) / (2 * n_values)

# Crear figura
fig = go.Figure()

# Curva inicial
fig.add_trace(go.Scatter(
    x=n_values,
    y=a_n,
    mode='lines+markers',
    name='aₙ = const + (n+1)/(2n)',  # subíndice con Unicode
    marker=dict(size=6)
))

# Línea del límite
fig.add_trace(go.Scatter(
    x=[1, n_max_init],
    y=[const_init + 0.5, const_init + 0.5],
    mode='lines',
    line=dict(dash='dash', color='red'),
    name=f'lim aₙ = {const_init + 0.5}'
))

# --- Sliders ---
n_slider_values = np.arange(10, 201, 5)
const_slider_values = np.arange(0, 10.1, 0.1)

# Slider n_max
steps_n = []
for n_max in n_slider_values:
    n_vals = np.arange(1, n_max+1)
    a_vals = const_init + (n_vals + 1)/(2*n_vals)
    steps_n.append(dict(
        method='update',
        label=str(n_max),
        args=[{'x':[n_vals, [1, n_max]], 'y':[a_vals, [const_init+0.5]*2]}]
    ))

# Slider constante
steps_const = []
for c in const_slider_values:
    a_vals = c + (n_values + 1)/(2*n_values)
    steps_const.append(dict(
        method='update',
        label=str(round(c,1)),
        args=[{'y':[a_vals, [c+0.5]*len(n_values)]},
              {'annotations':[dict(x=n_max_init/2, y=max(a_vals)+0.5, text=f"Gráfica de la sucesión aₙ", showarrow=False, font=dict(size=20))]}]
    ))

sliders = [
    dict(active=int((n_max_init-10)/5), currentvalue={"prefix":"n_max = "}, pad={"t":50}, steps=steps_n, x=0, xanchor='left', y=-0.15, len=0.48),
    dict(active=int(const_init*10), currentvalue={"prefix":"const = "}, pad={"t":50}, steps=steps_const, x=0.52, xanchor='left', y=-0.15, len=0.48)
]

# Layout
fig.update_layout(
    sliders=sliders,
    title="Gráfica de la sucesión aₙ",  # título con subíndice Unicode
    xaxis_title="n",
    yaxis_title="aₙ",
    template="plotly_white",
    margin=dict(b=150)
)

# Guardar HTML interactivo

gráfico sucesión#

#código para gráfica estática de la sucesión a_n
import matplotlib.pyplot as plt
import numpy as np

# Valores de n del 1 al 100
n_values = np.arange(1, 101)
# Definimos la sucesión a_n
a_n = 4 + (n_values + 1) / (2 * n_values)

# Graficar
plt.figure(figsize=(8, 5))
plt.plot(n_values, a_n, label=r'$a_n = 4 + \frac{n+1}{2n}$', color='blue', marker='o', markersize=4)
plt.axhline(y=4.5, color='red', linestyle='--', label=r'$\lim a_n = \frac{9}{2}$')

plt.title('Gráfica de la sucesión $a_n$')
plt.xlabel('n')
plt.ylabel(r'$a_n$')
plt.grid(True)
plt.legend()
plt.tight_layout()
plt.show()
\[ a_n = 4 + \frac{n+1}{2n} \]

Interpretación de la gráfica#

  1. Definición de la sucesión La sucesión está dada por

    \[ a_n = 4 + \frac{n+1}{2n}, \]

    donde \(n\) toma valores enteros positivos, por ejemplo desde \(n=1\) hasta \(n=100\).

  2. Comportamiento inicial Para valores pequeños de \(n\), el término \(\tfrac{n+1}{2n}\) es mayor que \(0.5\). Por ejemplo, para \(n=1\),

    \[ \frac{n+1}{2n} = \frac{2}{2} = 1, \]

    de modo que \(a_1 = 5\). Así, la sucesión comienza en un valor cercano a 5.

  3. Convergencia al límite A medida que \(n\) crece, \(\tfrac{n+1}{2n}\) se aproxima a \(\tfrac{1}{2}\). Por lo tanto,

    \[ \lim_{n\to\infty} a_n = 4 + \tfrac{1}{2} = \tfrac{9}{2} = 4.5. \]

    En la gráfica esto se aprecia como puntos que se acercan a la recta horizontal \(y=4.5\) (línea punteada roja).

  4. Tasa de convergencia La sucesión decrece suavemente hacia el límite desde arriba, y la diferencia \(|a_n - 4.5|\) se reduce conforme \(n\) aumenta.

  5. Conclusión La representación gráfica confirma que

    \[ \lim_{n\to\infty} a_n = \tfrac{9}{2}, \]

    y que la sucesión converge de manera monótonamente decreciente hacia ese valor.
